[행렬]Quadratic form

Derivative of a Quadratic Form

  1. Let \( \vec{a} = (a_1,a_2,\cdots,a_n)^T, \vec{\mu} = ( \mu_1, \mu_2, \cdots , \mu_n)^T \)

\( \begin{aligned} &{ \partial \vec{a}^T\vec{\mu} \over \partial \vec{\mu} } = { \partial \vec{\mu}^T\vec{a} \over \partial \vec{\mu} } = { \partial \over \partial \vec{\mu} }\sum_{i=1}^aa_iu_i \\\ &= [ { \partial \sum_{i=1}^aa_iu_i \over \partial \vec{\mu}_1 } \cdots { \partial \sum_{i=1}^aa_iu_i \over \partial \vec{\mu}_n } ] \\\ &= [ a_1, a_2, \cdots, a_n ] \\\ &= a^T \end{aligned} \)

  1. Let A be a nXn matrix \( a_{ij} \)
    \( \begin{aligned} &{ \partial \vec{\mu}^TA\vec{\mu} \over \partial \vec{\mu} } = { \partial \over \partial \vec{\mu} }[\sum_{i=1}^au_ia_{i1} \cdots \sum_{i=1}^au_ia_{in}]\begin{bmatrix} \mu_1 \\\ &\vdots \\\ \mu_n \end{bmatrix} \\\ &= { \partial \over \partial \vec{\mu} }\sum_{i=1}^na_{ij}\mu_i\mu_j \\\ &= \begin{bmatrix} 2a_{11}\mu_1+\sum_{j=2}^na_{ij}\mu{j}+\sum_{i=2}^na_{i1}\mu{i} \\\ \vdots \\\ 2a_{kk}\mu_k+\sum_{j=1}^na_{kj}\mu{j}+\sum_{j=1}^na_{ik}\mu{i} \\\ 2a_{nn}\mu_n+\sum_{j=1}^{n-1}a_{nj}\mu{j}+\sum_{j=1}^{n-1}a_{in}\mu{i} \end{bmatrix}^T \\\ &=sum_{j=1}^{n}a_{ij}\mu{j}+\sum_{i=1}^{n}a_{i1}\mu{i} \cdots \sum_{j=1}^{n}a_{nj}\mu{j}+\sum_{i=1}^{n}a_{in}\mu{i} \\\ &= [\sum_{j=1}^na_{ij}\mu{j}+ \cdots \sum_{j=1}^na_{nj}\mu{j}] + [ \sum_{i=1}^na_{i1}\mu{i}+ \cdots \sum_{i=1}^{n}a_{in}\mu{i}] \\\ &=\vec{\mu^T}A^T+\vec{\mu^T}A = \vec{\mu^T}(A+A^T) \end{aligned} \)

If A symmetric, \( 2\vec{\mu}^TA \)
