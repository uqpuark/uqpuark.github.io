  1. 리뷰(upper bound의 3가지 표현)
    $$ \begin{aligned} \alpha =\text{ least upper bound of E } &\Longleftrightarrow \begin{cases} 1.\ \alpha \text { upper bound of E } \\\ 2.\ \gamma < \alpha \rightarrow \gamma \text{ is not upper bound of E} \end{cases} \\\ &\Longleftrightarrow \begin{cases} 1.\ \alpha \text { upper bound of E } \\\ 2.\ \text{ if } \gamma < \alpha, \text{ then } \exists x \in E \text{ s.t } \gamma < x \end{cases} \\\ &\Longleftrightarrow \begin{cases} 1.\ \alpha \text { upper bound of E } \\\ 2.\ \forall \epsilon>0,\ \exists x \in E \text{ s.t. } \alpha-\epsilon < x \end{cases} \end{aligned} $$

3번째 형태를 나중에 많이 사용한다.

CH2 Basic Topology

실수를 좀 더 자세히 들여다 보기(countable/uncountable) + 거리가 있는 집단에 대한 기본적인 이야기

  1. Topology가 무엇인가?
  • 거리(metric, distance)를 이용해서 한 점에서 거리가 d보다 작은 집합을 정의한다.
  • 한 점을 포함하는 원(open set, open ball)을 기준으로 여러 이야기를 풀어 나가는 것이 topology
  1. Cantor
  • 대수적 수가 아닌 초월수의 존재 발견(Liouville)
  • Algebriac number는 R에 비해 굉장히 작다(둘다 무한집합이지만)
  • 무한집합의 대소를 비교하는 개념이 나옴 : countable/uncountable
  • 자연수, 유리수, Algebraic은 기본적으로 무한의 개념에서 크기가 같고, 실수는 굉장히 다르다.
  1. Countable(무한집합의 크기를 분류)
  • ‘집합의 크기가 같다’의 정의 : E -> F 함수가 존재 + 1대1대응(1-1, onto)
  • 예) 유리수에 순서를 부여해서 나열(출석부, 기본적으로 대각선 비슷하게 만듬) $$ Q = \begin{cases} 0 \\\ 1 &2 &3 &4 &5 &\cdots\\\ -1 &-2 &-3 &-4 & -5 & \cdots \\\ 1/2 & &3/2 & & 5/2 &\cdots \\\ -1/2 & &-3/2 & & -5/2 &\cdots \end{cases} \\\ \\\ = \{ 0, 1, -1, 2, 3, -2, \cdots \} $$
  • \( Q = \{ q_1, q_2, q_3, q_4, q_5, \cdots \} \\\ N = \{ 1, 2, 3, 4, 5, \cdots \} \)
    \( f : N \rightarrow Q, \text{ let } f(n)=q_n \Rightarrow f \text{ is 1-1 and onto } \)
    (원소의 의미는 전혀 생각하지 않고, 개수만 쳐다보자)
  • 약속 : \( \#(Q) = \#(N) = \aleph_0 \) (유리수의 개수는 자연수의 개수와 같다고 하자, 개수가 실수만큼 있는 것은 아니고 무한히 많음, Aleph number)
  • \( \#(N) \ne \#(R) = c \) (continuum, 연속체)

  1. def 2.1, 2.2에서 inverse 개념 $$ f^{-1}(E)= \{ x \in A | f(x) \in E \} \\\ f(A_1) = \{ f(x) | x \in A_1 \} \\\ \\\ \bigcup f^{-1}(E_j) = f^{-1}(\bigcup E_j) \\\ \bigcap f^{-1}(E_j) = f^{-1} (\bigcap E_j) $$

  2. def 2.3 equivalence relation

  • 2/4와 1/2는 서로 다르지만 뭔가 같은 의미가 있다 - 애매한 것들을 명확하게 해주는 개념이 equivalence relatiaon, 얘네들을 같다고 해주자.
    • 자기 자신은 같고
    • A ~ B 이면 B ~ A
    • A ~ B, B ~ C이면 A ~ C
  • 실수에서 쓰는 equal도 이러한 관계가 있는 겁니다.
  1. Cardinal number
  • \( f : A \rightarrow B, \text{ 1-1, onto } \)
    A와 B는 equivalent라고 부르고, A의 집합의 크기와 B의 집합의 크기가 같다.
    이 크기를 Cardinal number of A

\( \begin{aligned} Z &= \{ 0, \pm1, \pm2, \cdots \} \\\ &= \{ 0,1,-1,2,-2, \cdots \} \\\ &= \{ r_1, r_2, r_3, \cdots \} \\\ \#(Z) &= \#(N) = \text{ the cardinal number of N } \\\ \#(S) &= \text{ the cardinal number of S} \end{aligned} \)

  • 정수도 유리수 처럼 자연수랑 집합의 크기가 같다.
  • 무한 집합 A에서 B로 가는 1-1있는 순간 크기가 같다고 정의할 것이다.
