[예측애널리틱스] 1주차

1. Introduction (1주차-1)

개요

데이터 : 예측 모델을 구축하기 위해 필요한 것

  • Y : 예측하는 대상
  • 예측 모델링 : 예측 모델을 구축하는 과정

수치예측(discrete variable)과 범주예측(continuous variable)

  1. 수치예측(regression)
  • 주어진 데이터를 이용해서 Y ~ X의 관계를 최대한 잘 설명하는 모델을 구축하고, 새로운 데이터가 주어지면 모델을 사용하여 예측을 실행하는 단계

2단계 구조로 나눠서 보는 것을 강조한 것 같음

  1. 분류(범주예측, classification) : classification 안에 예측이라는 의미가 내포
  • 정상/비정상 등의 범주형 변수를 0/1등의 숫자로 코딩해서 표현한다.

Small/Medium/Large같은 경우, 즉 범주형 변수에 대소관계가 있는 경우는 어떻게 할 것인가?

단변량 시계열 예측

  • 앞선 내용과 다르게 X변수가 없고, 고려하는 데이터는 자기 자신(Y)의 과거 데이터
  • 시계열 데이터로 자기상관 모델링

시계열 내용을 좀 봐야 할 것 같다.

핵심 아이디어 : X와 Y의 관계(함수)를 찾는다

  • 원래 X를 주어진 함수에 대입해서 Y를 찾는게 해오던 문제들.
  • but 예측 모델링에서는 X, Y 데이터가 주어진 상태에서 함수를 찾는 것
  • epsilon이 우리가 X로 설명할 수 없는 에러(오차항)

에러는 : 기하적으로 x로 만들어지는 평면(2차원의 경우)에 직교

머신러닝 = 예측모델이라고 봐도 무방하다

  • parameter : 모수, 매개변수 (발음을 잘하자, 2번째 a에 강세)
  • 데이터가 주어졌을 때 모델의 파라미터를 찾는 행위(학습, training)

파라미터 추정

  1. Loss function과 Cost function의 차이
    손실함수 : 하나의 관측치로 정의된 함수 비용함수 : 손실함수들이 결합된 형태(더하던, 평균을 내던)
  • 엄격하게 분리해서 쓰는 건 아니고, 대부분 혼합해서 사용한다.
  1. 오차의 합은 상쇄가 되어 0이 나온다(이론적으로 증명 가능)
  • 오차의 차이를 보는게 중요하기 때문에 제곱이나 절대값을 준다.

각각 L2(least squre), L1(least absolute) loss

  1. 비용함수를 최소로 하는 parameter(w1, w2)를 찾자
  • 추정 전의 w1, w2는 값이 정해지지 않았다.
  • 추정 후의 w1, w2는 값이 모델에 의해 하나로 정해짐(hat을 붙여줌)

estimator(공식, 추정량)를 이용하여 estimate(추정치)를 구하는 것 실제 데이터를 대입하기 전에는 확률변수이다.

  1. 모델의 종류
  • 데이터가 주어졌을때, 어떤 모델이 적절할지 결정 해야한다. 그 후 알고리즘을 통해 파라미터를 추정한다.

머신러닝 모델 학습 과정 요약

  1. (데이터 전처리)
  2. 모델 결정하기 (Y를 표현하기 위한 X들의 조합 방식을 결정)
  3. 모델을 구성하는 파라미터 찾기 - 학습(training)
  • 가지고 있는 데이터(X, Y)를 이용하여 실제 참값과 최대한 같게(cost function) 나오도록!

기계학습(ML) -> 예측 모델링

  • 아직까지 기계는 스스로 학습이 불가능하다. 가능한 인공지능은 아직까지 없다.

  • 핵심은 알고리즘 인데 그건 인간이 개발한다.

  • 인간은 단기기억력이 약하고, 단순 연산이 약한데,

  • 기계는 강하다. 대신, 창의력이 없어서 알고리즘 개발 불가, 구현은 잘함

Machine Learned by computer languages to perform the algorithm created by human

  • 인간이 컴퓨터 언어를 통해 기계를 학습 시킨다.

실습

쥬피터 노트북 사용한다!
